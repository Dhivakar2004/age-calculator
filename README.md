# 🎂 Age Calculator

A simple and responsive Age Calculator web application that calculates a user's exact age in years, months, and days based on the date of birth entered.

## 🚀 Live Demo

https://your-netlify-link.netlify.app/

## 📌 Features

- Calculate exact age
- Displays years, months, and days
- User-friendly interface
- Responsive design
- Instant calculation
- Input validation

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (ES6)


2. Open the project folder.

3. Open `index.html` in your browser.
